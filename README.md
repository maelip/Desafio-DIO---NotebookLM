# Desafio-DIO---NotebookLM

Miniguia – Juros Simples vs. Juros Compostos

Este repositório documenta o desenvolvimento de um miniguia de estudo criado no NotebookLM,
utilizando IA generativa como ferramenta de aprendizagem ativa.

Objetivo
Comparar juros simples e juros compostos de forma introdutória, com base em fontes abertas
e organização do conhecimento apoiada por IA.

Ferramentas Utilizadas
- NotebookLM
- Fontes abertas (PDF e texto)
- GitHub (documentação do projeto)

Estrutura do Projeto
- docs/: descrição e metodologia
- fontes/: materiais utilizados no estudo
- notebooklm/: organização do miniguia no NotebookLM
- prompts/: prompts reutilizáveis para estudo

Acesso ao Miniguia
O miniguia foi desenvolvido diretamente no NotebookLM.
https://notebooklm.google.com/notebook/662358f6-afca-4894-a6e5-8ddac08d3e03
